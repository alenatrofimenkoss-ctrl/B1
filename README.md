# Polski B1 — Przygotowanie do egzaminu

Прогрессивное веб-приложение (PWA) для подготовки к государственному экзамену
по польскому языку, уровень B1. Работает офлайн, устанавливается на главный
экран iPhone через Safari → «Поделиться» → «На экран Домой».

## Структура

- `index.html` — разметка и дизайн-система
- `app.js` — навигация, рендер экранов, движок тестов, прогресс (localStorage)
- `data_grammar.js` — грамматика B1 (7 тем: падежи, род/число, прилагательные, виды и времена глагола, числительные, предлоги/местоимения, сложные предложения)
- `data_skills_exams.js` — заглушки для разделов Czytanie / Słuchanie / Pisanie / Mówienie и 15 вариантов экзамена (наращиваются поэтапно)
- `manifest.json`, `sw.js`, `icon-*.png` — PWA-манифест, офлайн-кэш, иконки
